---
description: Manual debugging workflow for application errors
---

1. **Reproduce**: Identify the exact steps to trigger the error.
2. **Trace**: Check the browser console and network tab.
3. **Logs**: Check backend logs or `ai_changelog.md` for recent changes.
4. **Isolate**: Comment out recent changes to see if they are the cause.
5. **Fix & Verify**: Apply the fix and test against the reproduction steps.

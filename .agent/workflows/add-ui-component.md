---
description: How to add a new UI component using Shadcn
---

1. **Add Primitive**: Run `// turbo` `npx shadcn-ui@latest add [component-name]`.
2. **Customize**: Tailor the component in `components/ui/[component-name].tsx` to match GeekyZindagi branding.
3. **Usage**: Import and use in functional components.
4. **Animation**: Wrap or integrate with `framer-motion` for transitions.

---
description: End-to-end workflow for implementing a new feature
---

1. **Research**: Read `.agent/memory/architecture.md` and `.agent/memory/codebase-map.md`.
2. **Draft**: Create an implementation plan.
3. **Execute**: Build the data layer, then logic, then UI.
4. **Refine**: Apply animations and polish aesthetics.
5. **Document**: Update `ai_changelog.md` and relevant memory maps.
